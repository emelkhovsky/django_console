Lesson 3 - Navigation
На этом уроке мы с вами познакомимся с командами Linux для навигации

#history

Команда history показывает ранее введённые пользователем команды.

Попробуйте ввести:

history

#clear

Команда clear необходима для полной очистки окна терминала. Она удаляет все ранее выведенные сообщения.

Попробуйте ввести:

clear

Чтобы изменения вступили в силу, перезагрузите страницу.

#locate

Команда locate используется для поиска файла в системе Linux.

Эта команда полезна, когда вы не знаете, где файл сохранен или фактическое имя файла. Использование аргумента -i с командой помогает игнорировать регистр (не имеет значения, является ли он прописным или строчным).
Например, если вам нужен файл со словом в названии«hello», вы вводите «locate -i hello» и видите список всех файлов в вашей системе Linux, содержащих слово «hello».

Если вам необходимо найти два слова, вы можете разделить их звездочкой (*). Например, чтобы найти файл, содержащий слова «hello» и «this», вы можете использовать команду «locate -i * hello * this».
Попробуйте ввести:

locate -i filename

Если у вас был файл «filename.txt», то вы увидите путь до него.

#cd

Используйте эту команду, чтобы перейти из текущего каталога в указанный.

Если вы просто наберете «cd» и нажмете кнопку Отправить, вы попадете в домашний каталог. Чтобы указать другую папку для работы с файлами, нужно ввести: «cd адрес_папки»
Но есть один нюанс. Представьте, что у вас есть папка с именем «Raspberry Pi». В этом случае, когда вы вводите «cd Raspberry Pi», оболочка примет второй аргумент команды как другой, поэтому вы получите сообщение об ошибке, говорящее о том, что каталог не существует.

Здесь вы можете использовать обратную косую черту, то есть: «cd Raspberry\ Pi».
Чтобы поднятся на уровень выше, вы можете набрать «cd ..». Вызов с двумя точками возвращает на уровень вверх относительно текущего каталога.

Вызов с тире «cd – » возвращает к предыдущему каталогу.

Создайте каталог «for_cd » через команду mkdir.
Попробуйте перейти в этот каталог:

cd for_cd

С помощью команды pwd проверьте, удалось ли вам поменять местоположение.

Для удобства вы можете вернуться обратно к себе вв папку с помощью команды cd ..

#zcat / zless / zmore

Это утилиты для просмотра сжатых файлов.

С помощью zcat можно найти и увидеть список сжатых файлов

zless выводит их списком постранично с возможностью листать вперёд и назад zmore — только вперёд.

Пока что у вас нет сжатых файлов, но мы уверены, что в будущем вы с ними справитесь.

#;

Это, конечно, не команда, но в данном уроке было бы полезно узнать о таком операторе. Точка с запятой — ещё один оператор для последовательного запуска нескольких команд в одной строке. Если нужно, чтобы команды выполнялись параллельно, используйте оператор так: (команда_№1 &); (команда_№2 &); (команда_№3 &)
Попробуйте создать файл, заполнить и вывести его:

(touch file1.txt &); (echo good >> file1.txt &); (cat file1.txt &);
