Lesson 5 - Text
В этом уроке мы вас научим командам для работы с текстом. Для этого создайте файл filetext.txt, filetext2.txt и запишите в них несколько одинаковых строчек текста(кроме одной).

#head/tail

Команда head выводит несколько первых строк из файла (голова), а tail выдает несколько последних строк (хвост).
По умолчанию каждая утилита выводит десять строк. Но это можно изменить с помощью опции -n.

Ещё один полезный параметр -f, это сокращение от follow (следовать). Утилита постоянно выводит изменения в файле на экран. Например, если вы хотите, чтобы лог (журнал) файла отображался в режиме реального времени используйте команду «tail -nf».
Попробуйте ввести:

head -n 1 filetext.txt

Вы увидите первую строчку вашего файла.

#sort

Команда выполняет сортировку текстовой информации по заданным показателям. Дополнительные переменные позволяют:

-n - сортировать строки текста по числовому значению;

-du - по размеру;

-r - применять обратную сортировку;

-u - удалять дубликаты при сортировке;

Вот несколько принципов, по которым команда sort сортирует строки:

-Строки с цифрами размещаются выше других строк

-Строки, начинающиеся с букв нижнего регистра размещаются выше

-Сортировка выполняется в соответствии алфавиту

-Строки сначала сортируются по алфавиту, а уже вторично по другим правилам

Попробуйте вывывести текст вашего файла с помощью команды cat.

А теперь давайте введем команду:

sort filetext.txt

Вы увидите новую, отсортированную версию вашего файла. Но будьте осторожны, сортировка НЕ меняет содержимое файла.

#wc

Утилита командной строки Linux для анализа файла.

На выходе вы получите кол-во строк, слов, символов. Программа также может принимать параметры для анализа отдельных значений. Например:

-с - отобразить размер объекта в байтах и др.
Попробуйте ввести:

wc filetext.txt

Вы увидите информацию о файле.

#diff

Данная команда производит построчное сравнение двух текстовых файлов, выводя различия между ними. Причём выводятся только строки, в которых обнаружены отличия. Алгоритм работы: «diff файл_№1 файл_№2»
Параметры, с которыми можно использовать команду:

-q - выводить только отличия файлов;

-s - выводить только совпадающие части;

-с - выводить нужное количество строк после совпадений и др.
В выводе утилиты кроме, непосредственно, отображения изменений, выводит строку в которой указывается, в какой строчке и что было сделано. Для этого используются такие символы:

a - добавлена;

d - удалена;

c - изменена.
Попробуйте ввести:

diff filetext.txt filetext2.txt

В результате, вы можете, например, увидеть:

-Стрелки < и >, где < указывает принадлежность строки, стоящей после стрелочки к 1ому файлу, а > - ко 2ому.

-Некоторые параметры, указывающие на различия в файлах(например 1,3d0 обозначает, что строки с первой по третью должны быть удалены из начала первого файла, чтобы получился 2ой файл)
