Lesson 2 - File control
На этом уроке мы с вами познакомимся с командами Linux для управления файлами

#pwd

В прошлом уроке вы уже успели познакомиться с этой командой, которая отображает ваш текущий каталог!

Изначально вы находитесь в каталоге c именем вашего логина. На протяжении всех уроков вы будете работать в этой папке, не выходя за ее пределы.
Если хотите, попробуйте еще раз:

pwd

После ввода команды вы увидите ответ консоли в таком виде:

home/ваш_логин/

home - это каталог, в котором находятся каталоги всех пользователей. В рамках данных уроков вы не сможете покинуть свою папку, перейти в папку home, папки других пользователей и др.

#mkdir

Используя эту команду, вы можете создать папку или каталог. Если вы хотите создать пустой каталог «name», вы можете ввести «mkdir name».

Попробуйте ввести:

mkdir name

Если вы не получили сообщение об ошибке, все в порядке, папка была успещно создана.

#ls

Давайте проверим, создан ли у вас новый каталог name, используя команду ls.
ls - утилита для просмотра содержимого каталогов. Используйте эту команду, чтобы узнать, какие файлы расположены в каталоге, в котором вы находитесь (сейчас вы находитесь в каталоге ваш_логин)
Если указать путь, то вы увидите содержимое конечного каталога.

Вы можете увидеть все скрытые файлы, используя команду «ls -a».

С помощью команды «ls -l» вы сможете увидеть вывод в виде списка с более подробной информацией. Команда «ll» заменяет команду «ls -l», но может работать не на всех дистрибутивах Linux.
Введите:

ls -l

чтобы проверить, что ваш каталог создан. Если все правиьно, то вы увидите, что появилась новая строчка с названием «name», это и есть наш каталог.

#touch

С помощью этой команды вы можете создать новый файл в каталоге. Если он еще не существует, он создаст новый пустой. Если файл с таким именем уже существует, он обновит временные метки файла.

Введите:

touch filename.txt

Мы можем проверить, что файл создан с помощью уже знакомой команды -

ls

Если в списке файлов, вы нашли файл с именем filename, то все сделано правильно.

#echo

Это простая команда, которая просто выводит свои аргументы на дисплей. Если ввести просто echo some text, то на экране вы увидите текстовое сообщение «some text»

Опция -e с echo позволяет вам интерпретировать escape-символы:
/c - удалить перевод строки;
/t - горизонтальная табуляция;
/v - вертикальная табуляция;
/b - удалить предыдущий символ;
/n - перевод строки;
/r - символ возврата каретки в начало строки; Попробуйте ввести:

echo –е "Some\n text"

Если все прошло удачно, то в консоли вы увидите текст!
Но самое актуальное для нас это то, что с помощью этой команды можно записать любое текстовое сообщение в файл, для этого необходимо ввести: echo «текст сообщения» >> «файл» Например, запишем слово «text» в файл «filename.txt»

Для этого вводим:

echo text >> filename.txt

#cat

Используйте команду cat для отображения содержимого файла. Когда команда используется с одним текстовым файлом (cat путь_к_файлу), она отображает его содержимое в окне терминала.

Если указать более одного файла(cat путь_к_файлу_1 путь_к_файлу_2), она склеит их.
Если ввести (cat путь_к_файлу_1> новый_файл), она объединит содержимое указанных файлов в новый файл. Если файл пустой, то команда ничего не выведет. Попробуйте ввести:

cat filename.txt

Если все прошло удачно, то ниже вы увидите текст файла!

#cp

Используйте команду cp для копирования файла. Он принимает два аргумента: первый — это местоположение файла, который нужно скопировать, второй - куда копировать.
Место назначения копирования может быть файлом или директорией.
Попробуйте создать файл filename2.txt с помощью команды touch. Например, нужно скопировать файл file.txt в файл filename2.txt.

Для этого вводим:

cp filename.txt filename2.txt

Проверьте содержимое файла filename2.txt с помощью команды cat. Если все верно, то в нем будет то же самое, что и в файле filename.txt.
Чтобы скопировать файл в директорию, можно воспользоваться командой cp имя_файла имя_директории/

#mv

Используйте команду mv, чтобы переименовывать файлы. Он принимает два аргумента: имя файла и новое имя файла. Например, давайте переименуем файл «filename.txt» в «newfilename.txt»

Для этого попробуйте ввести:

mv filename.txt newfilename.txt

Вы можете проверить поменялось ли имя файла с помощью команды ls.

#file

Используя эту команду, можно посмотреть, к какому типу относится файл. В Linux файлы не обязаны всегда иметь расширения для того, чтобы с ними работать. Поэтому пользователю иногда трудно определить, что за файл перед ним. Эта маленькая утилита решает проблему.

Синтаксис: file [option] [filename]
Можно использовать параметр -b, который нужен для отображения только типа файла в кратком режиме.

Попробуйте ввести:

file -b newfilename.txt

Если все правильно, то вы увидите вывод «ASCII text»

#rm

Используйте команду rm для удаления файлов и каталогов. Аккуратнее, rm удаляет файлы не в корзину, из которой потом всё можно будет восстановить, а полностью стирает.

Действия оператора rm необратимы.
Команда «rm -r имя_каталога» используется, чтобы удалить только каталог.

Чтобы удалить один файл, используйте команду rm, за которой следует имя файла: «rm имя_файла»

Если файл защищен от записи, вам будет предложено подтвердить ее(rm: remove write-protected regular empty file 'имя_файла'?). Чтобы удалить файл, введите y(yes) или n(no) и нажмите кнопку Отправить.
В противном случае, если файл не защищен от записи, он будет удален без запроса.

Чтобы удалить сразу несколько файлов, используйте команду rm, за которой следуют имена файлов, разделенные пробелом: rm имя_файла1 имя_файла2 имя_файла3

Вы также можете использовать подстановочный знак ( *) и регулярные выражения для соответствия нескольким файлам.
Например, чтобы удалить все файлы с расширением .pdf в текущем каталоге, используйте следующую команду: rm *.pdf

Как удалять каталоги?

Для удаления пустого каталога используйте опцию -d. rm -d имя_каталога.

Чтобы удалить непустые каталоги и все файлы внутри них, используйте параметр -r: rm -r dirname. Если каталог или файл в каталоге защищены от записи, вам будет предложено подтвердить удаление.
Чтобы удалить непустые каталоги и все файлы без запроса, используйте параметры -r и -f: rm -rf имя_каталога

Чтобы удалить сразу несколько каталогов, используйте команду rm, за которой следуют имена каталогов, разделенные пробелом: rm -r имя_каталога1 имя_каталога2 имя_каталога3
Давайте создадим каталог «dir_for_rm» с помощью команды mkdir: mkdir dir_for_rm. Вы можете проверить наличие файла с помощью команды ls -l.

Теперь давайте попробуем удалить каталог:

rm -r dir_for_rm

С помощью команды ls -l убедитесь, что его нет в списке файлов и каталогов.
